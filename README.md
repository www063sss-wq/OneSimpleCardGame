一直想自己制作一款游戏，之前玩三国杀想制作一款卡牌游戏

# 设计哲学
1. 必须要有个故事背景，这个故事背景必须宏大，以宇宙，平行世界为空间维度，这样才能兼容不同的人物发展
2. 这幅卡牌必须能线下能玩，不能是只能线上玩
3. 这幅卡牌的创作过程应当开源

1. 这个故事应当是架空的，卡牌的设计会有很多技能，在故事里面反应的则是力量本源，如果想要技能丰富多彩，在故事中应该遵循一个根本力量，然后转化出来很多子力量，子力量相互组合，可以形成不同的力量体系。
2. 人物的战斗是讲究策略的，卡牌多数是回合制，策略是基于时间维度的，只有一个方向，如果故事中的时间可以逆转，在回合制的设计上应当适当体现出来。
3. 卡牌可以肯定有人物牌，拟定一个场景牌，不同的场景下有不同的效果，还有剧情牌，剧情牌会触发一些效果，比如更改游戏结局。
4. 人物牌有生命属性，背包属性（装备），背包属性提供了一个策略组，还有能力属性，能力属性也提供了一个策略组
5. 除了人物牌，还有场景牌，背包牌，剧情牌，战斗牌。战斗牌是另一个策略组，现实生活中战斗是空间立体的，如何在牌上反应出来呢？斗地主给了我很好的想法，你可以一张主牌+多张副牌一并打出表示一个立体的多样化策略。
6. 比如一张主牌规定了某个主属性战斗策略，副牌详细刻画了这个主属性的战斗策略，比如是魔法伤害，物理伤害等。
7. 可以造成伤害的方式有很多种，不一定是魔法物理这两种
8. 加入一个状态牌，表示一个人物正在处于何种状态
9. 如果这幅牌是8个人，我如何设计能确保每个人的等待时间是可接受的
10. 故事中的每个人在不同阶段是不同样子，所以可以同一个人有不同的卡牌与之对应，就是同名不同属性
11. 如果剧情中某个人败给某个人，那么在技能上应当符合1v1情况下，对战胜率要低于对方
12. 这个故事中应当有一个主角，主角随着成长能力多变
13. 骰子的话用抬牌代替也可以
14. 战斗牌加入花色点数两个属性，人物牌加入地域种族职业属性，背包牌也加入花色点数属性。
15. 人与人之间引入距离的概念，装备引入腐朽，被击毁属性，场景引入可以被改造的属性，
